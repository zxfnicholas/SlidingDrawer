# SlidingDrawer
高仿QQ侧滑,抽屉式菜单
===================
一.前言：
-------------

>  - 此项目是根据网上的demo整理而来，是个不错的项目。
>  - 根据Android原生的DrawerLayou实现。

二.软件功能：
-------------

>  - 1.可以左右侧滑
>  - 2.可以仿QQ式，也可以以普通的抽屉式呈现
